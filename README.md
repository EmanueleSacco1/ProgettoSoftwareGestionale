## Descrizione del Progetto

Questo progetto è un'applicazione gestionale sviluppata in Python che permette di gestire prenotazioni, prodotti, fatture, eventi in agenda e tariffe. Utilizza il modulo `pickle` per salvare e caricare i dati in file locali.

### Funzionalità Principali

- **Prenotazioni**: Creazione, visualizzazione e gestione delle prenotazioni dei clienti.
- **Prodotti**: Creazione, visualizzazione, modifica ed eliminazione dei prodotti.
- **Fatture**: Creazione e visualizzazione delle fatture.
- **Agenda**: Creazione e visualizzazione degli eventi in agenda.
- **Tariffe**: Creazione e visualizzazione delle tariffe.

### Struttura del Progetto

- `Backend/agenda.py`: Gestione degli eventi in agenda.
- `Backend/fattura.py`: Gestione delle fatture.
- `Backend/main.py`: Punto di ingresso dell'applicazione.
- `Backend/pickle_manager.py`: Funzioni per salvare e caricare i dati utilizzando `pickle`.
- `Backend/prenotazione.py`: Gestione delle prenotazioni.
- `Backend/prodotto.py`: Gestione dei prodotti.
- `Backend/tariffa.py`: Gestione delle tariffe.
